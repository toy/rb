build(&blk)

Creates an ad-hoc Rack application within the context of a Rack::Builder. @yield The block to be used to instantiate the `Rack::Builder`.

@example 

    get { halt build { use Rack::ContentLength; run proc { |env| Rack::Response.new("Hello!").finish } } }

run(app = nil, &blk)

Runs a rack application. You must either use `app` or `blk`. @param [call] app The application to call. @yield [env] The block to yield to

@example 

    get { halt run proc { |env| Renee::Core::Response.new("Hello!").finish } }
