parse_options!(args, *extras, &block)

Use Slop to parse the arguments given.

@param [Array] args The options are stripped out by Slop. @param [*Symbol] extras Extra features you want returned. @param [&Block] used to add custom arguments to Slop.

@option [Extra] :method_object Returns a method object.

@return Slop::Options iff you don’t pass any extras. @return [Array] If you do pass extras, an array is returned where the first argument is the Slop::Options object, and the remainder are the extras you requested in order.
