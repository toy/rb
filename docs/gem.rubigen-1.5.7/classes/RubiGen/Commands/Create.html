file(relative_source, relative_destination, file_options = {}, &block)

Copy a file from source to destination with collision checking.

The file_options hash accepts :chmod and :shebang and :collision options. :chmod sets the permissions of the destination file: 

  file 'config/empty.log', 'log/test.log', :chmod => 0664

:shebang sets the #!/usr/bin/ruby line for scripts 

  file 'bin/generate.rb', 'script/generate', :chmod => 0755, :shebang => '/usr/bin/env ruby'

:collision sets the collision option only for the destination file: 

  file 'settings/server.yml', 'config/server.yml', :collision => :skip

Collisions are handled by checking whether the destination file exists and either skipping the file, forcing overwrite, or asking the user what to do.

220:       def file(relative_source, relative_destination, file_options = {}, &block)
221:         # Determine full paths for source and destination files.
222:         source              = source_path(relative_source)
223:         destination         = destination_path(relative_destination)
224:         destination_exists  = File.exist?(destination)
225: 
226:         # If source and destination are identical then we're done.
227:         if destination_exists and identical?(source, destination, &block)
228:           return logger.identical(relative_destination)
229:         end
230: 
231:         # Check for and resolve file collisions.
232:         if destination_exists
233: 
234:           # Make a choice whether to overwrite the file.  :force and
235:           # :skip already have their mind made up, but give :ask a shot.
236:           choice = case (file_options[:collision] || options[:collision]).to_sym #|| :ask
237:             when :ask   then force_file_collision?(relative_destination, source, destination, file_options, &block)
238:             when :force then :force
239:             when :skip  then :skip
240:             else raise "Invalid collision option: #{options[:collision].inspect}"
241:           end
242: 
243:           # Take action based on our choice.  Bail out if we chose to
244:           # skip the file; otherwise, log our transgression and continue.
245:           case choice
246:             when :force then logger.force(relative_destination)
247:             when :skip  then return(logger.skip(relative_destination))
248:             else raise "Invalid collision choice: #{choice}.inspect"
249:           end
250: 
251:         # File doesn't exist so log its unbesmirched creation.
252:         else
253:           logger.create relative_destination
254:         end
255: 
256:         # If we're pretending, back off now.
257:         return if options[:pretend]
258: 
259:         # Write destination file with optional shebang.  Yield for content
260:         # if block given so templaters may render the source file.  If a
261:         # shebang is requested, replace the existing shebang or insert a
262:         # new one.
263:         File.open(destination, 'wb') do |dest|
264:           dest.write render_file(source, file_options, &block)
265:         end
266: 
267:         # Optionally change permissions.
268:         if file_options[:chmod]
269:           FileUtils.chmod(file_options[:chmod], destination)
270:         end
271: 
272:         # Optionally add file to subversion or git
273:         system("svn add #{destination}") if options[:svn]
274:       end

template(relative_source, relative_destination, template_options = {})

Generate a file using an ERuby template. Looks up and evaluates a template by name and writes the result.

The ERB template uses explicit trim mode to best control the proliferation of whitespace in generated code. <%- trims leading whitespace; -%> trims trailing whitespace including one newline.

A hash of template options may be passed as the last argument. The options accepted by the file are accepted as well as :assigns, a hash of variable bindings. Example: 

  template 'foo', 'bar', :assigns => { :action => 'view' }

Template is implemented in terms of file. It calls file with a block which takes a file handle and returns its rendered contents.
