alt_escape(cmdline)

Escape special characters used in most unix shells to use it, eg. with system().

This differs from Ruby’s escape in that it does not escape shell variables, e.g. $0.

37:   def run(argv, opts)
38:     argv = (String === argv ? shellwords(argv) : argv.to_a.dup)
39:     args = []
40:     while argv.any?
41:       item = argv.shift
42:       flag = opts[item]
43:       if flag
44:         # Work around lambda semantics in 1.8.7.
45:         arity = [flag.arity, 0].max
46:         # Raise if there are not enough parameters
47:         # available for the flag.
48:         if argv.size < arity
49:           raise ArgumentError
50:         end
51:         # Call the lambda with N items from argv,
52:         # where N is the lambda's arity.
53:         flag.call(*argv.shift(arity))
54:       else
55:         # Collect the items that don't correspond to
56:         # flags.
57:         args << item
58:       end
59:     end
60:     args
61:   end
