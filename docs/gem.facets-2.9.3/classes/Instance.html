to_h(at=false)

Return instance variables with values as a hash. 

  class X
    def initialize(a,b)
      @a, @b = a, b
    end
  end

  x = X.new(1,2)

  x.instance.to_h  #=> { :a=>1, :b=>2 }

update(hash)

Set instance variables given a hash. 

  instance.update('@a'=>1, '@b'=>2)
  @a   #=> 1
  @b   #=> 2

Also, +@+ sign is not neccessary. 

  instance.update(:a=>1, :b=>2)
  @a   #=> 1
  @b   #=> 2
This method is also aliased as assign
