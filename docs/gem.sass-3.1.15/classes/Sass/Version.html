Instance Public methods
version()

Returns a hash representing the version of Sass. The `:major`, `:minor`, and `:teeny` keys have their respective numbers as Fixnums. The `:name` key has the name of the version. The `:string` key contains a human-readable string representation of the version. The `:number` key is the major, minor, and teeny keys separated by periods. If Sass is checked out from Git, the `:rev` key will have the revision hash. For example: 

    {
      :string => "2.1.0.9616393",
      :rev    => "9616393b8924ef36639c7e82aa88a51a24d16949",
      :number => "2.1.0",
      :major  => 2, :minor => 1, :teeny => 0
    }

If a prerelease version of Sass is being used, the `:string` and `:number` fields will reflect the full version (e.g. `“2.2.beta.1“`), and the `:teeny` field will be `-1`. A `:prerelease` key will contain the name of the prerelease (e.g. `“beta”`), and a `:prerelease_number` key will contain the rerelease number. For example: 

    {
      :string => "3.0.beta.1",
      :number => "3.0.beta.1",
      :major => 3, :minor => 0, :teeny => -1,
      :prerelease => "beta",
      :prerelease_number => 1
    }

@return [{Symbol => String/Fixnum}] The version hash
