Class Public methods
new(h, s, l)

Initialize an HSL colour where: 

    0 <= h <= 360
    0 <= s <=   1
    0 <= l <=   1

Saturation and luminance values outside these ranges will be clipped. Hue values will be mapped to a circle, so that e.g. -20 becomes 340.
