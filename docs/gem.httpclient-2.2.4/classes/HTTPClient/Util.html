uri_part_of(uri, part)

Returns true if the given 2 URIs have a part_of relationship.

  • the same scheme
  • the same host String (no host resolution or IP-addr conversion)
  • the same port number
  • target URI’s path starts with base URI’s path.

argument_to_hash(args, *field)

Keyword argument to hash helper.

args:given arguments.
*field:a list of arguments to be extracted.

Returns hash which has defined keys. When a Hash given, returns it including undefined keys. When an Array given, returns a Hash which only includes defined keys.

Source: show 

    # File lib/httpclient/util.rb, line 61

62:       return nil if args.empty?
63:       if args.size == 1 and Hash === args[0]
64:         h = args[0]
65:         if field.any? { |f| h.key?(f) }
66:           return h
67:         end
68:       end
69:       h = {}
70:       field.each_with_index do |e, idx|

72:       end

keyword_argument(args, *field)

Keyword argument helper.

args:given arguments.
*field:a list of arguments to be extracted.

You can extract 3 arguments (a, b, c) with: 

  include Util
  def my_method(*args)
    a, b, c = keyword_argument(args, :a, :b, :c)
    ...
  end
  my_method(1, 2, 3)
  my_method(:b => 2, :a = 1)

instead of; 

  def my_method(a, b, c)
    ...
  end

Source: show 

    # File lib/httpclient/util.rb, line 44
44:     def keyword_argument(args, *field)
45:       if args.size == 1 and Hash === args[0]
46:         h = args[0]
47:         if field.any? { |f| h.key?(f) }
48:           return h.values_at(*field)
49:         end
50:       end
51:       args
52:     end
