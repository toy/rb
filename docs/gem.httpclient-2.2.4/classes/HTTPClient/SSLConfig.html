Class HTTPClient::SSLConfig < Object

  • lib/httpclient/ssl_config.rb

Represents SSL configuration for HTTPClient instance. The implementation depends on OpenSSL.

Trust Anchor Control

SSLConfig loads ‘httpclient/cacert.p7s’ as a trust anchor (trusted certificate(s)) with add_trust_ca in initialization time. This means that HTTPClient instance trusts some CA certificates by default, like Web browsers. ‘httpclient/cacert.p7s’ is created by the author and included in released package.

‘cacert.p7s’ is automatically generated from JDK 1.6.

You may want to change trust anchor by yourself. Call clear_cert_store then add_trust_ca for that purpose.

Attributes
[R] client_cert
OpenSSL::X509::Certificate:certificate for SSL client authenticateion.

nil by default. (no client authenticateion)

[R] client_key
OpenSSL::PKey::PKey:private key for SSL client authentication.

nil by default. (no client authenticateion)

[R] verify_mode

A number which represents OpenSSL’s verify mode. Default value is OpenSSL::SSL::VERIFY_PEER | OpenSSL::SSL::VERIFY_FAIL_IF_NO_PEER_CERT.

[R] verify_depth

A number of verify depth. Certification path which length is longer than this depth is not allowed.

[R] verify_callback

A callback handler for custom certificate verification. nil by default. If the handler is set, handler.call is invoked just after general OpenSSL’s verification. handler.call is invoked with 2 arguments, ok and ctx; ok is a result of general OpenSSL’s verification. ctx is a OpenSSL::X509::StoreContext.

[R] timeout

SSL timeout in sec. nil by default.

[R] options

A number of OpenSSL’s SSL options. Default value is OpenSSL::SSL::OP_ALL | OpenSSL::SSL::OP_NO_SSLv2

[R] ciphers

A String of OpenSSL’s cipher configuration. Default value is ALL:!ADH:!LOW:!EXP:!MD5:+SSLv2:@STRENGTH See ciphers(1) man in OpenSSL for more detail.

[R] cert_store

OpenSSL::X509::X509::Store used for verification. You can reset the store with clear_cert_store and set the new store with cert_store=.

302:     def sample_verify_callback(is_ok, ctx)
303:       unless is_ok
304:         depth = ctx.error_depth
305:         code = ctx.error
306:         msg = ctx.error_string
307:         warn("at depth #{depth} - #{code}: #{msg}") if $DEBUG
308:         return false
309:       end
310: 
311:       cert = ctx.current_cert
312:       self_signed = false
313:       ca = false
314:       pathlen = nil
315:       server_auth = true
316:       self_signed = (cert.subject.cmp(cert.issuer) == 0)
317: 
318:       # Check extensions whatever its criticality is. (sample)
319:       cert.extensions.each do |ex|
320:         case ex.oid
321:         when 'basicConstraints'
322:           /CA:(TRUE|FALSE), pathlen:(\d+)/ =~ ex.value
323:           ca = ($1 == 'TRUE')
324:           pathlen = $2.to_i
325:         when 'keyUsage'
326:           usage = ex.value.split(/\s*,\s*/)
327:           ca = usage.include?('Certificate Sign')
328:           server_auth = usage.include?('Key Encipherment')
329:         when 'extendedKeyUsage'
330:           usage = ex.value.split(/\s*,\s*/)
331:           server_auth = usage.include?('Netscape Server Gated Crypto')
332:         when 'nsCertType'
333:           usage = ex.value.split(/\s*,\s*/)
334:           ca = usage.include?('SSL CA')
335:           server_auth = usage.include?('SSL Server')
336:         end
337:       end
338: 
339:       if self_signed
340:         warn('self signing CA') if $DEBUG
341:         return true
342:       elsif ca
343:         warn('middle level CA') if $DEBUG
344:         return true
345:       elsif server_auth
346:         warn('for server authentication') if $DEBUG
347:         return true
348:       end

351:     end

359:     def load_cacerts(cert_store)
360:       [
361:         [DIST_CERT, 'cacert.p7s'],
362:         [DIST_CERT_SHA1, 'cacert_sha1.p7s']
363:       ].each do |cert_str, ca_file|
364:         file = File.join(File.dirname(__FILE__), ca_file)

368:           store = X509::Store.new

372:             return

374:         end
375:       end
376:       warn("cacerts loading failed")
