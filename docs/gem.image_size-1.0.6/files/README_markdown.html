# image_size

measure image size using pure Ruby formats: PCX, PSD, XPM, TIFF, XBM, PGM, PBM, PPM, BMP, JPEG, PNG, GIF, SWF

## Download

The latest version of image_size can be found at github.com/toy/image_size

## Installation 

    gem install image_size

## Simple Example 

    ruby "image_size"
    ruby "open-uri"

    open("http://www.rubycgi.org/image/ruby_gtk_book_title.jpg", "rb") do |fh|
      p ImageSize.new(fh.read).size
    end
