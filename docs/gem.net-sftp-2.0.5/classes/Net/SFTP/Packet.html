Class Net::SFTP::Packet < Net::SSH::Buffer

  • lib/net/sftp/packet.rb
  • lib/net/sftp/packet.rb

A specialization of the Net::SSH::Buffer class, which simply auto-reads the type byte from the front of every packet it represents.
