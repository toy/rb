normalize_path(path)

Normalizes URI path.

Strips off trailing slash and ensures there is a leading slash. 

  normalize_path("/foo")  # => "/foo"
  normalize_path("/foo/") # => "/foo"
  normalize_path("foo")   # => "/foo"
  normalize_path("")      # => "/"
