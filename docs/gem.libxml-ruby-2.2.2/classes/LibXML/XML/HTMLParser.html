XML::HTMLParser.file(path) → XML::HTMLParser XML::HTMLParser.file(path, :encoding => XML::Encoding::UTF_8, :options => XML::HTMLParser::Options::NOENT) → XML::HTMLParser

Creates a new parser by parsing the specified file or uri.

You may provide an optional hash table to control how the parsing is performed. Valid options are: 

 encoding - The document encoding, defaults to nil. Valid values
            are the encoding constants defined on XML::Encoding.
 options - Parser options.  Valid values are the constants defined on
           XML::HTMLParser::Options.  Mutliple options can be combined
           by using Bitwise OR (|).
