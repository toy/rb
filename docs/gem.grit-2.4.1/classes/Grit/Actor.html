output(time)

Outputs an actor string for Git commits. 

  actor = Actor.new('bob', 'bob@email.com')
  actor.output(time) # => "bob <bob@email.com> UNIX_TIME +0700"

time - The Time the commit was authored or committed.

Returns a String.
