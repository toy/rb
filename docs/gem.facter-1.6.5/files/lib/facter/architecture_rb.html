Fact: architecture

Purpose: 

  Return the CPU hardware architecture.

Resolution: 

  On OpenBSD, Linux and Debian's kfreebsd, use the hardwaremodel fact.
  Gentoo and Debian call "x86_86" "amd64".
  Gentoo also calls "i386" "x86".
