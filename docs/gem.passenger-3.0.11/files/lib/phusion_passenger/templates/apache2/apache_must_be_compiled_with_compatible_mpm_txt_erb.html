WARNING: Apache doesn’t seem to be compiled with the ‘prefork’ or ‘worker’ MPM

Passenger has only been tested on Apache with the ‘prefork’ and the ‘worker’ MPM. Your Apache installation is compiled with the ’<%= @current_mpm %>’ MPM. We recommend you to abort this installer and to recompile Apache with either the ‘prefork’ or the ‘worker’ MPM.

Press Ctrl-C to abort this installer (recommended). Press Enter if you want to continue with installation anyway.
