Nginx with Passenger support was successfully installed.

Please edit your Nginx configuration file<% if @config_file %> (probably <%= @config_file %>)<% end %>, and set the passenger_root and passenger_ruby configuration options in the ‘http’ block, like this: 

  <b>http {
      ...
      passenger_root <%= @passenger_root %>;
      passenger_ruby <%= @ruby %>;
      ...
  }</b>

After you (re)start Nginx, you are ready to deploy any number of Ruby on Rails applications on Nginx.

Press ENTER to continue.
