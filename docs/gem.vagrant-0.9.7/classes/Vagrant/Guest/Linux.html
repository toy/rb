expanded_guest_path(guestpath)

Determine the real guest path. Since we use a `sudo` shell everywhere else, things like ’~’ don’t expand properly in shared folders. We have to `echo` here to get that path.

@param [String] guestpath The unexpanded guest path. @return [String] The expanded guestpath

79:         @vm.channel.execute("printf #{guestpath}") do |type, data|
80:           if type == :stdout
