y( object, *objects )

Prints any supplied objects out in YAML. Intended as a variation on +Kernel::p+. 

  S = Struct.new(:name, :state)
  s = S['dave', 'TX']
  y s

produces: 

  --- !ruby/struct:S
  name: dave
  state: TX
