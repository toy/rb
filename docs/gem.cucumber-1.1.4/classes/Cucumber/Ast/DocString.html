Class Cucumber::Ast::DocString < String

  • lib/cucumber/ast/doc_string.rb

Represents an inline argument in a step. Example: 

  Given the message
    """
    I like
    Cucumber sandwich
    """

The text between the pair of """ is stored inside a DocString, which is yielded to the StepDefinition block as the last argument.

The StepDefinition can then access the String via the to_s method. In the example above, that would return: "I likenCucumber sandwich"

Note how the indentation from the source is stripped away.
