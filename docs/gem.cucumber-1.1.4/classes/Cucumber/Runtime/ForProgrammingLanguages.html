table(text_or_table, file=nil, line_offset=0)

Returns a Cucumber::Ast::Table for text_or_table, which can either be a String: 

  table(%{
    | account | description | amount |
    | INT-100 | Taxi        | 114    |
    | CUC-101 | Peeler      | 22     |
  })

or a 2D Array: 

  table([
    %w{ account description amount },
    %w{ INT-100 Taxi        114    },
    %w{ CUC-101 Peeler      22     }
  ])
