get_guard_class(name, fail_gracefully=false)

Tries to load the Guard main class. This transforms the supplied Guard name into a class name:

  • `guardname` will become `Guard::Guardname`
  • `dashed-guard-name` will become `Guard::DashedGuardName`
  • `underscore_guard_name` will become `Guard::UnderscoreGuardName`

When no class is found with the strict case sensitive rules, another try is made to locate the class without matching case:

  • `rspec` will find a class `Guard::RSpec`

@param [String] name the name of the Guard @param [Boolean] fail_gracefully whether error messages should not be printed @return [Class, nil] the loaded class

Source: show 

     # File lib/guard.rb, line 456
456:     def get_guard_class(name, fail_gracefully=false)
457:       name        = name.to_s
458:       try_require = false
459:       const_name  = name.gsub(/\/(.?)/) { "::#{ $1.upcase }" }.gsub(/(?:^|[_-])(.)/) { $1.upcase }
460:       begin
461:         require "guard/#{ name.downcase }" if try_require
462:         self.const_get(self.constants.find { |c| c.to_s == const_name } || self.constants.find { |c| c.to_s.downcase == const_name.downcase })
463:       rescue TypeError
464:         unless try_require
465:           try_require = true
466:           retry
467:         else

469:         end
470:       rescue LoadError => loadError
471:         unless fail_gracefully
472:           UI.error "Could not load 'guard/#{ name.downcase }' or find class Guard::#{ const_name.capitalize }"
