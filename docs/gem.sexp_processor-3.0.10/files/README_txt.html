SexpProcessor

rdoc :parsetree.rubyforge.org/sexp_processor
  • rubyforge.org/projects/parsetree/

DESCRIPTION:

sexp_processor branches from ParseTree bringing all the generic sexp processing tools with it. Sexp, SexpProcessor, Environment, etc... all for your language processing pleasure.

FEATURES/PROBLEMS:

  • Includes SexpProcessor and CompositeSexpProcessor.
    • Allows you to write very clean filters.

SYNOPSIS:

  class MyProcessor < SexpProcessor
    def initialize
      super
      self.strict = false
    end
    def process_lit(exp)
      val = exp.shift
      return val
    end
  end

REQUIREMENTS:

  • rubygems

INSTALL:

  • sudo gem install sexp_processor
