add_attribute(attribute)

Adds attribute if not already there. If it is (as method(s) or attribute), updates the comment if it was empty.

The attribute is registered only if it defines a new method. For instance, attr_reader :foo will not be registered if method foo exists, but attr_accessor :foo will be registered if method foo exists, but foo= does not.

add_class(class_type, given_name, superclass = '::Object')

Adds a class named given_name with superclass.

Both given_name and superclass may contain ’::’, and are interpreted relative to the self context. This allows handling correctly examples like these: 

  class RDoc::Gauntlet < Gauntlet
  module Mod
    class Object   # implies < ::Object
    class SubObject < Object  # this is _not_ ::Object

Given class Container::Item RDoc assumes Container is a module unless it later sees class Container. add_class automatically upgrades given_name to a class in this case.

350:     # superclass +nil+ is passed by the C parser in the following cases:
351:     # - registering Object in 1.8 (correct)
352:     # - registering BasicObject in 1.9 (correct)
353:     # - registering RubyVM in 1.9 in iseq.c (incorrect: < Object in vm.c)
354:     #
355:     # If we later find a superclass for a registered class with a nil
356:     # superclass, we must honor it.
357: 
358:     # find the name & enclosing context
359:     if given_name =~ /^:+(\w+)$/ then
360:       full_name = $1
361:       enclosing = top_level
362:       name = full_name.split(/:+/).last
363:     else
364:       full_name = child_name given_name
365: 
366:       if full_name =~ /^(.+)::(\w+)$/ then
367:         name = $2
368:         ename = $1
369:         enclosing = RDoc::TopLevel.classes_hash[ename] ||
370:                     RDoc::TopLevel.modules_hash[ename]
371:         # HACK: crashes in actionpack/lib/action_view/helpers/form_helper.rb (metaprogramming)
372:         unless enclosing then
373:           # try the given name at top level (will work for the above example)
374:           enclosing = RDoc::TopLevel.classes_hash[given_name] || RDoc::TopLevel.modules_hash[given_name]
375:           return enclosing if enclosing
376:           # not found: create the parent(s)
377:           names = ename.split('::')
378:           enclosing = self
379:           names.each do |n|
380:             enclosing = enclosing.classes_hash[n] ||
381:                         enclosing.modules_hash[n] ||
382:                         enclosing.add_module(RDoc::NormalModule, n)
383:           end
384:         end
385:       else
386:         name = full_name
387:         enclosing = self
388:       end
389:     end
390: 
391:     # fix up superclass
392:     superclass = nil if full_name == 'BasicObject'
393:     superclass = nil if full_name == 'Object' and defined?(::BasicObject)
394:     superclass = '::BasicObject' if
395:       defined?(::BasicObject) and full_name == 'Object'
396:

414: 
415:       upgrade_to_class mod, RDoc::NormalClass, mod.parent if mod
416: 
417:       # e.g., Object < Object

419:     end
420:

422: 
423:     if klass then
424:       # if TopLevel, it may not be registered in the classes:
425:       enclosing.classes_hash[name] = klass
426:

428:       if superclass then
429:         existing = klass.superclass
430:         existing = existing.full_name unless existing.is_a?(String) if existing
431:         if existing.nil? ||

433:           klass.superclass = superclass
434:         end
435:       end
436:     else
437:       # this is a new class
438:       mod = RDoc::TopLevel.modules_hash.delete full_name
439:

441:         klass = upgrade_to_class mod, RDoc::NormalClass, enclosing
442: 
443:         klass.superclass = superclass unless superclass.nil?

446:

448:                                       RDoc::TopLevel.classes_hash)
449:       end
450:     end

each_section()

Iterator for each section’s contents sorted by title. The section, the section’s constants and the sections attributes are yielded. The constants and attributes collections are sorted.

To retrieve methods in a section use methods_by_type with the optional section parameter.

NOTE: Do not edit collections yielded by this method
