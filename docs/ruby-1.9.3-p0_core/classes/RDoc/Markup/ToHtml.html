handle_special_HYPERLINK(special)

special is a potential link. The following schemes are handled:

mailto::Inserted as-is.
http::Links are checked to see if they reference an image. If so, that image gets inserted using an <img> tag. Otherwise a conventional <a href> is used.
link::Reference to a local file relative to the output directory.
