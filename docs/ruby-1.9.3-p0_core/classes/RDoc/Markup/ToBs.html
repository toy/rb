Class RDoc::Markup::ToBs < RDoc::Markup

  • lib/rdoc/markup/to_bs.rb

Outputs RDoc markup with hot backspace action! You will probably need a pager to use this output format.

This formatter won’t work on 1.8.6 because it lacks String#chars.

67:   def convert_string string
68:     return string unless string.respond_to? :chars # your ruby is lame
69:     return string unless @in_b or @in_em
70:     chars = if @in_b then
71:               string.chars.map do |char| "#{char}\b#{char}" end
72:             elsif @in_em then
73:               string.chars.map do |char| "_\b#{char}" end
