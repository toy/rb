add_line_numbers(src)

Prepend src with line numbers. Relies on the first line of a source code listing having: 

  # File xxxxx, line dddd

If it has, line numbers are added an ’, line dddd’ is removed.

Source: show 

     # File lib/rdoc/generator/markup.rb, line 87
 87:   def add_line_numbers(src)
 88:     return unless src.sub!(/\A(.*)(, line (\d+))/, '\1')
 89:     first = $3.to_i - 1
 90:     last  = first + src.count("\n")
 91:     size = last.to_s.length
 92: 
 93:     line = first
 94:     src.gsub!(/^/) do
 95:       res = if line == first then
 96:               " " * (size + 1)
 97:             else
 98:               "<span class=\"line-num\">%2$*1$d</span> " % [size, line]
 99:             end
100: 
101:       line += 1
102:       res
103:     end
104:   end
