Class Fixnum::Bignum::Rational < Object

  • lib/mathn.rb

When mathn is required Rational is changed to simplify the use of Rational operations.

Normal behaviour: 

  Rational.new!(1,3) ** 2 # => Rational(1, 9)
  (1 / 3) ** 2            # => 0

require ‘mathn’ behaviour: 

  (1 / 3) ** 2            # => 1/9
