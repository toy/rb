# random_text

github.com/toy/random_text/tree/master

## DESCRIPTION:

Class to make usage of vesna.yandex.ru/ and lipsum.com/ easier

## SYNOPSIS: 

    include RandomText

    Lorem.word
    Lorem.words(5)

    Lorem.uniq_words(10)

    Lorem.sentence
    Lorem.sentences(5)

    Lorem.paragraph
    Lorem.paragraphs(5)

Same can be done for Vesna

    lorem 3 paragraphs
