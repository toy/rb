Icebox : Caching for HTTParty

Cache responses in HTTParty models [github.com/jnunemaker/httparty]

Usage

  class Foo
    include HTTParty
    include HTTParty::Icebox
    cache :store => 'file', :timeout => 600, :location => MY_APP_ROOT.join('tmp', 'cache')
  end

Modeled after Martyn Loughran’s APICache [github.com/newbamboo/api_cache] and Ruby On Rails’s caching [api.rubyonrails.org/classes/ActiveSupport/Cache.html]

Author: Karel Minarik [www.karmi.cz]
