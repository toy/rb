The cache missed or a reload is required. Forward the request to the backend and determine whether the response should be stored. This allows conditional / validation requests through to the backend but performs no caching of the response when the backend returns a 304.

lookup()

Try to serve the response from cache. When a matching cache entry is found and is fresh, use it as the response without forwarding any request to the backend. When a matching cache entry is found but is stale, attempt to validate the entry with the backend using conditional GET. When no matching cache entry is found, trigger miss processing.

Source: show 

     # File lib/rack/cache/context.rb, line 159

160:       if @request.no_cache? && allow_reload?
161:         record :reload
162:         fetch
163:       else

validate(entry)

Validate that the cache entry is fresh. The original request is used as a template for a conditional GET request with the backend.

Source: show 

     # File lib/rack/cache/context.rb, line 188
188:     def validate(entry)
189:       # send no head requests because we want content
190:       @env['REQUEST_METHOD'] = 'GET'
191: 
192:       # add our cached last-modified validator to the environment
193:       @env['HTTP_IF_MODIFIED_SINCE'] = entry.last_modified
194: 
195:       # Add our cached etag validator to the environment.
196:       # We keep the etags from the client to handle the case when the client
197:       # has a different private valid entry which is not cached here.
198:       cached_etags = entry.etag.to_s.split(/\s*,\s*/)
199:       request_etags = @request.env['HTTP_IF_NONE_MATCH'].to_s.split(/\s*,\s*/)
200:       etags = (cached_etags + request_etags).uniq
201:       @env['HTTP_IF_NONE_MATCH'] = etags.empty? ? nil : etags.join(', ')
202: 
203:       response = forward
204: 
205:       if response.status == 304
206:         record :valid
207: 
208:         # Check if the response validated which is not cached here
209:         etag = response.headers['ETag']
210:         return response if etag && request_etags.include?(etag) && !cached_etags.include?(etag)
211: 
212:         entry = entry.dup
213:         entry.headers.delete('Date')
214:         %w[Date Expires Cache-Control ETag Last-Modified].each do |name|
215:           next unless value = response.headers[name]
216:           entry.headers[name] = value
217:         end
218: 
219:         # even though it's empty, be sure to close the response body from upstream
220:         # because middleware use close to signal end of response
221:         response.body.close if response.body.respond_to?(:close)
222: 
223:         response = entry
224:       else
225:         record :invalid
226:       end

228:       store(response) if response.cacheable?

230:       response
