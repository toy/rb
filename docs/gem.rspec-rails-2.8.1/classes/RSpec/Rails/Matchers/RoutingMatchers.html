be_routable()

Passes if the route expression is recognized by the Rails router based on the declarations in `config/routes.rb`. Delegates to `RouteSet#recognize_path`.

@example

You can use route helpers provided by rspec-rails. 

    {:get =>  "/a/path"}.should be_routable
    {:post => "/another/path"}.should be_routable
    {:put => "/yet/another/path"}.should_not be_routable

route_to(*expected)

Delegates to `assert_recognizes`. Supports short-hand controller/action declarations (e.g. `“controlleraction“`).

@example 

    { :get => "/things/special" }.should route_to(
      :controller => "things",
      :action     => "special"
    )

    { :get => "/things/special" }.should route_to("things#special")

@see api.rubyonrails.org/classes/ActionDispatch/Assertions/RoutingAssertions.html#method-i-assert_recognizes
