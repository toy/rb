When installed as a plugin this is loaded automatically.

When installed as a gem, you need to add 

 require 'new_relic/recipes'

to your deploy.rb

Defined deploy:notify_rpm which will send information about the deploy to New Relic. The task will run on app servers except where no_release is true. If it fails, it will not affect the task execution or do a rollback.
