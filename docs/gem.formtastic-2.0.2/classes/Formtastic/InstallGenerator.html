Class Formtastic::InstallGenerator < Rails::Generators::Base

  • lib/generators/formtastic/install/install_generator.rb

Copies formtastic.css to public/stylesheets/ (Rails 3.0.x only) and a config initializer to config/initializers/formtastic.rb (all Rails versions).

@example
