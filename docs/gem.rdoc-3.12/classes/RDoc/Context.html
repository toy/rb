Class RDoc::Context < RDoc::CodeObject

add_attribute(attribute)

Adds attribute if not already there. If it is (as method(s) or attribute), updates the comment if it was empty.

The attribute is registered only if it defines a new method. For instance, attr_reader :foo will not be registered if method foo exists, but attr_accessor :foo will be registered if method foo exists, but foo= does not.

add_class(class_type, given_name, superclass = '::Object')

Adds a class named given_name with superclass.

Both given_name and superclass may contain ’::’, and are interpreted relative to the self context. This allows handling correctly examples like these: 

  class RDoc::Gauntlet < Gauntlet
  module Mod
    class Object   # implies < ::Object
    class SubObject < Object  # this is _not_ ::Object

Given class Container::Item RDoc assumes Container is a module unless it later sees class Container. add_class automatically upgrades given_name to a class in this case.

243:   def add_class class_type, given_name, superclass = '::Object'
244:     # superclass +nil+ is passed by the C parser in the following cases:
245:     # - registering Object in 1.8 (correct)
246:     # - registering BasicObject in 1.9 (correct)
247:     # - registering RubyVM in 1.9 in iseq.c (incorrect: < Object in vm.c)
248:     #
249:     # If we later find a superclass for a registered class with a nil
250:     # superclass, we must honor it.
251: 
252:     # find the name & enclosing context
253:     if given_name =~ /^:+(\w+)$/ then
254:       full_name = $1
255:       enclosing = top_level
256:       name = full_name.split(/:+/).last
257:     else
258:       full_name = child_name given_name
259:

261:         name = $2
262:         ename = $1
263:         enclosing = RDoc::TopLevel.classes_hash[ename] ||
264:                     RDoc::TopLevel.modules_hash[ename]
265:         # HACK: crashes in actionpack/lib/action_view/helpers/form_helper.rb (metaprogramming)
266:         unless enclosing then
267:           # try the given name at top level (will work for the above example)
268:           enclosing = RDoc::TopLevel.classes_hash[given_name] || RDoc::TopLevel.modules_hash[given_name]
269:           return enclosing if enclosing
270:           # not found: create the parent(s)
271:           names = ename.split('::')
272:           enclosing = self
273:           names.each do |n|
274:             enclosing = enclosing.classes_hash[n] ||
275:                         enclosing.modules_hash[n] ||
276:                         enclosing.add_module(RDoc::NormalModule, n)
277:           end
278:         end
279:       else
280:         name = full_name
281:         enclosing = self
282:       end
283:     end
284: 
285:     # fix up superclass
286:     superclass = nil if full_name == 'BasicObject'
287:     superclass = nil if full_name == 'Object' and defined?(::BasicObject)
288:     superclass = '::BasicObject' if
289:       defined?(::BasicObject) and full_name == 'Object'

308:

313:     end
314:

318:       # if TopLevel, it may not be registered in the classes:
319:       enclosing.classes_hash[name] = klass
320:

322:       if superclass then
323:         existing = klass.superclass
324:         existing = existing.full_name unless existing.is_a?(String) if existing
325:         if existing.nil? ||

327:           klass.superclass = superclass
328:         end
329:       end
330:     else
331:       # this is a new class
332:       mod = RDoc::TopLevel.modules_hash.delete full_name
333:

336: 
337:         klass.superclass = superclass unless superclass.nil?

340: 
341:         enclosing.add_class_or_module(klass, enclosing.classes_hash,
342:                                       RDoc::TopLevel.classes_hash)
343:       end
344:     end

each_section()

Iterator for each section’s contents sorted by title. The section, the section’s constants and the sections attributes are yielded. The constants and attributes collections are sorted.

To retrieve methods in a section use methods_by_type with the optional section parameter.

NOTE: Do not edit collections yielded by this method
