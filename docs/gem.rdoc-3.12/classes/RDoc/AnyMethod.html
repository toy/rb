add_line_numbers(src)

Prepend src with line numbers. Relies on the first line of a source code listing having: 

  # File xxxxx, line dddd

If it has, line numbers are added an ’, line dddd’ is removed.

Source: show 

     # File lib/rdoc/generator/markup.rb, line 88
 88:   def add_line_numbers(src)
 89:     return unless src.sub!(/\A(.*)(, line (\d+))/, '\1')
 90:     first = $3.to_i - 1
 91:     last  = first + src.count("\n")
 92:     size = last.to_s.length
 93: 
 94:     line = first
 95:     src.gsub!(/^/) do
 96:       res = if line == first then
 97:               " " * (size + 1)
 98:             else
 99:               "<span class=\"line-num\">%2$*1$d</span> " % [size, line]
100:             end
101: 
102:       line += 1
103:       res
104:     end
105:   end
