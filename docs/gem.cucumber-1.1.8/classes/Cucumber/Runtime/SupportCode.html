load_programming_language(ext)

Loads and registers programming language implementation. Instances are cached, so calling with the same argument twice will return the same instance.

Source: show 

    # File lib/cucumber/runtime/support_code.rb, line 71
71:       def load_programming_language(ext)
72:         return @language_map[ext] if @language_map[ext]
73:         programming_language_class = constantize("Cucumber::#{ext.capitalize}Support::#{ext.capitalize}Language")
74:         programming_language = programming_language_class.new(@runtime_facade)

182:         if ext = File.extname(step_def_file)[1..-1]
183:           return nil if @unsupported_programming_languages.index(ext)
